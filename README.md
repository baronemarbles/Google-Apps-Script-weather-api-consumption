# Google-Apps-Script-weather-api-consumption
Script para o consumo da "Free Weather API" com a finalidade de alimentar a planilha do Google Sheets, formatar os dados para o formato de graus célsius e em português, e automatização da seleção dos arquivos de imagem e gif, que são pareados com os  dados climáticos, utilizando as fórmulas do Google Sheets.

A proposta é o VMIX consumir a API do google para que se alimenta desta planilha e utilize os dados climáticos para a exibição do clima nas cidades de cobertura PNT do Jornal "VTV da gente" da SBT.
